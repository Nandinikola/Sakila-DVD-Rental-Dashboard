# Sakila DVD Rental Dashboard

## Project Overview
This Power BI dashboard analyzes the Sakila DVD Rental database.

## Dashboard Features
- Total Revenue
- Total Rentals
- Revenue by Category
- Rental Distribution by Store
- Top 10 Films by Rentals
- Monthly Rental Trend
- Customer Distribution by Country
- Interactive Slicers
